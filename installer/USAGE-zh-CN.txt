【macOS · 安装】
═══════════════════════════════════════════════════════════════
  1. 从 GitHub Releases 下载 CodexHelper-x.x.x-macos.dmg
  2. 双击打开 DMG
  3. 推荐：把 Codex Helper 拖到「应用程序」
  4. 首次打开看菜单栏右上角（无 Dock 图标）

  若提示「已损坏，无法打开」——不是文件坏了，是 macOS 安全拦截。
  任选一种方式：

  方式 A（推荐，避开 /Applications 权限问题）
    打开「终端」，逐行执行：

    mkdir -p ~/Applications
    cp -R "/Volumes/Codex Helper/Codex Helper.app" ~/Applications/
    xattr -cr "$HOME/Applications/Codex Helper.app"
    open "$HOME/Applications/Codex Helper.app"

  方式 B（装到系统应用程序文件夹）
    系统设置 → 隐私与安全性 → 完全磁盘访问权限 → 打开「终端」
    关掉终端再开，然后：

    sudo xattr -dr com.apple.quarantine "/Applications/Codex Helper.app"
    sudo codesign --force --deep --sign - "/Applications/Codex Helper.app"

  方式 C
    在 Finder 里右键 Codex Helper →「打开」→ 再点「打开」

  API Key 写入 ~/.codex/.env 。诊断：codex-helper doctor

  若图标上有 🚫 禁止符号（无法在此 Mac 上打开）：
    多半是 CPU 架构不匹配。请用最新版 DMG（含 arm64 + x86_64 通用包），或终端检查：

    file "/Applications/Codex Helper.app/Contents/MacOS/codex-helper"
    uname -m

    应看到 universal 或与本机 uname -m 一致（Apple 芯片 = arm64，Intel = x86_64）。
    安装前先退出旧版（菜单栏 → 退出），卸载旧 App，弹出侧边栏里多余的 Codex Helper 磁盘。

【Windows · 环境要求】
═══════════════════════════════════════════════════════════════
  · Windows 10 / 11（64 位）
  · 已安装 OpenAI Codex Desktop，并能正常登录
  · 各模型厂商的 API Key（DeepSeek、千问等官网申请）

【绿色版 · 首次使用】
  1. 解压本 zip 到任意目录（路径尽量不要含中文空格）
  2. 双击 codex-helper.exe 启动（首次使用也可先执行：.\codex-helper.exe init）
  3. 看任务栏托盘出现 Codex Helper 图标
  4. 右键托盘 →「设置…」→ 选厂商、填 API Key → 保存
  5. 完全退出并重新打开 Codex Desktop

【日常使用】
  · 切换厂商：托盘 →「切换模型 · xxx」
  · 切换具体型号：托盘 →「设置…」→ 热门模型
  · 诊断问题：.\codex-helper.exe doctor
  · 恢复 OpenAI 官方：托盘 → 更多 →「切换回 OpenAI 官方」

【开机自启（可选）】
  Win+R 输入 shell:startup，把 codex-helper.exe 的快捷方式放进去，
  目标加上参数：start

【配置位置】
  · %USERPROFILE%\.codex-helper\     本工具配置与 Key
  · %USERPROFILE%\.codex\            Codex 配置（工具会自动写入代理）

【本地代理端口】
  · 固定为 127.0.0.1:25543（Codex 访问 http://127.0.0.1:25543/v1）
  · 自检：浏览器打开 http://127.0.0.1:25543/health
  · 若 Codex 连不上：先确认托盘在运行，再托盘 →「重新同步配置」

【卸载绿色版】
  托盘 → 退出；删除解压目录即可。配置在 .codex-helper 里，需手动删。
